## Installation

`docker pull frapsoft/alm`

## Examples

### run alm in the local folder

`docker run -v $(pwd):/app -p 4444:4444 frapsoft/alm .`

### run alm with volume

`docker run -v webapp:/app -p 4444:4444 frapsoft/alm .`
